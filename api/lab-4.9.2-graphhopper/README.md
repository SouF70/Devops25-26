# Lab 4.9.2 – Integrate a REST API in a Python Application

## Doel
In dit lab werd een REST API geïntegreerd in een Python applicatie.
De Graphhopper API werd gebruikt om locaties te geocoden en routes te berekenen.

## Uitvoering
- REST API calls uitgevoerd met Python (`requests`)
- JSON-responses geparsed
- Latitude en longitude uit API-data gehaald
- Routes berekend tussen twee locaties
- Afstand en reistijd weergegeven
- User input gebruikt voor locaties en voertuigtype

## Resultaat
De Python applicatie kan op basis van gebruikersinput een route berekenen
en toont de afstand en reistijd in de terminal.
