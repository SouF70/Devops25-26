## Context
Dit experiment werd gemaakt in het kader van het vak **DevOps – Application Deployment and Security**  
en is gebaseerd op:

- DevNet Associate – Instructor Training  
- **Presentation 2 – API Calls**

Het doel van deze opdracht is zelfstandig experimenteren met REST API-calls in Python,
zoals gedemonstreerd tijdens de lessen.

---

## Gebruikte API
**GraphHopper REST API**  
https://www.graphhopper.com/

---

## Beschrijving
In dit experiment wordt met een Python-script een REST API-call uitgevoerd naar de
GraphHopper API.  
De API berekent de afstand en de geschatte reistijd tussen twee locaties.

De API-response wordt ontvangen in JSON-formaat en verwerkt in Python.

---

## Aangetoonde concepten
- REST API-gebruik
- API key authenticatie
- HTTP GET request
- JSON-verwerking in Python
- Resultaat weergeven in de terminal

Deze concepten sluiten rechtstreeks aan bij **Presentation 2 – API Calls**.
