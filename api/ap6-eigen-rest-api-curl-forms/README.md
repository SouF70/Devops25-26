# AP6 – Eigen REST-API experiment met curl en forms

## Context
Dit experiment werd uitgevoerd in het kader van het vak
DevOps en is gebaseerd op Presentation 2 – API Calls.

## Beschrijving
In dit experiment werd een REST API ontwikkeld in Python
met Flask. De API laat toe om boeken toe te voegen via
zowel een HTML webformulier als via curl-commando’s.

Beide methodes gebruiken dezelfde API-endpoint.

## Gebruikte technieken
- Python
- Flask
- REST API
- HTML forms
- curl
- JSON
- HTTP statuscodes

## Uitvoering

python3 app.py
