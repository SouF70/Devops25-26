# AP5 – Eigen REST-API experiment met curl

## Context
Dit experiment werd uitgevoerd in het kader van het vak
DevOps en is gebaseerd op Presentation 2 – API Calls.

## Beschrijving
In dit experiment werd een eigen REST API ontwikkeld in Python
met behulp van Flask. De API beheert een lijst van boeken en laat toe
om deze op te vragen en uit te breiden via HTTP-requests.

De API wordt getest met het command-line tool `curl`,
zoals vereist in AP5.

## Gebruikte technieken
- Python
- Flask
- REST API
- HTTP methods (GET, POST)
- curl
- JSON
- HTTP statuscodes

## Uitvoering

python3 api.py
