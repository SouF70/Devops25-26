# Pf1 – Flask-experiment (lab 6.3.6)

## Context
Dit experiment werd uitgevoerd in het kader van het vak DevOps en is gebaseerd op lab 6.3.6.

## Doel
Een eenvoudige Flask webapplicatie bouwen als basis voor microservices.

## Uitvoering
- Flask geïnstalleerd in een virtual environment
- Flask app ontwikkeld met één route
- HTML template gebruikt via Flask templates
- Applicatie lokaal getest via browser

## Resultaat
De Flask applicatie is bereikbaar op http://127.0.0.1:5000.
