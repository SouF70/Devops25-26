# PF2 – Flask Logon Page Experiment

## Beschrijving
Dit experiment demonstreert een eenvoudige loginpagina gebouwd met Flask.  
De focus ligt op het verwerken van formulierinvoer (username en password) en het tonen van feedback aan de gebruiker.

Deze oefening is gebaseerd op Lab 6.5.10 – Explore the Evolution of Password Methods.

---

## Functionaliteit
- Flask webapplicatie
- Loginformulier met:
  - Username
  - Password
- Verwerking van POST-requests
- Foutmelding bij ongeldige login (*Invalid username/password*)
- Webserver draait lokaal via Flask

> Dit experiment implementeert geen echte gebruikersdatabase.  
> Het doel is inzicht krijgen in loginflows en basis-authenticatie, niet in beveiligde productie-authenticatie.

---

## Gebruikte technologieën
- Python 3
- Flask
- HTML (templates)
- Python virtual environment (venv)
