# Pv1 – Python Virtual Environment (Lab-experiment)

## Context
Dit experiment kadert binnen het vak DevOps
en is gebaseerd op Presentation 6 – Python Working Environments.

## Doel
Het doel van dit experiment is het aanmaken en gebruiken van een
Python virtual environment (venv) om afhankelijkheden geïsoleerd te beheren.

## Uitvoering
- Een virtual environment werd aangemaakt met `python3 -m venv`
- De environment werd geactiveerd
- Het Python package `requests` werd geïnstalleerd met `pip`
- De environment werd nadien gedeactiveerd

## Besluit
Python virtual environments maken het mogelijk om afhankelijkheden
per project gescheiden te houden en vormen een belangrijke basis
voor deployment en automatisatie.
