# Pv2 – Python Virtual Environment (Deployment)

## Doel
Aantonen dat een Python applicatie gedeployed en uitgevoerd kan worden binnen een virtual environment.

## Stappen
- Virtual environment aangemaakt met `python3 -m venv`
- Environment geactiveerd
- Dependencies geïnstalleerd via `requirements.txt`
- Applicatie uitgevoerd vanuit de venv

## Resultaat
De applicatie werd succesvol uitgevoerd en maakte een HTTP-request via het `requests` package.

## Conclusie
Virtual environments zijn essentieel voor consistente deployments en dependency management.
